Add an application

We will now go through the steps of creating an OpenFin launcher for a web application of your choosing, the first step is to provide a name and the url, this process will generate an app.json file that will be used to create a launcher and configure the OpenFin Runtime.

You are now ready to create an installer, follow these steps to create a shortcut.

The app.json file will be used on application start to determine the runtime settings, it can also be used to generate installers you can distribute.

We recommend that you read more about the app.json file, the OpenFin runtime version manager (RVM) and deployment strategies on our documentation site.
